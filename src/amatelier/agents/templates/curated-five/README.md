# curated-five

The default roster amatelier ships with. Five named workers plus the three admin-side roles every roundtable needs.

## Workers

- **elena** (Claude Sonnet) — synthesis and architecture. Strong at pulling threads together and proposing structured approaches.
- **marcus** (Claude Sonnet) — challenge and exploit detection. Looks for the failure the proposal didn't account for.
- **clare** (Claude Haiku) — concise structural analysis. Fast passes, clean outlines, minimal hedge.
- **simon** (Claude Haiku) — triage and fix sequencing. Orders work by risk and dependency, not by what's easiest.
- **naomi** (Gemini Flash) — cross-cutting and novel framing. Brings angles the others miss because they think alike.

## Admin-side

- **admin** — curation, quality evaluation, dispute resolution. Spawned post-RT when the digest flags issues (failed scoring, Byzantine behavior, skill curation).
- **judge** — scores each roundtable on novelty, accuracy, impact, challenge. 0-3 plus a 10 for grand insights.
- **therapist** — exit interviews, memory updates, behavioral coaching. Runs after each RT to keep agents honest with themselves.

## What this template is for

General-purpose roundtables. Works well on code review, design review, architecture debates, research briefings, and any prompt where "propose, critique, refine" is the natural shape. The model mix keeps cost reasonable: Haiku carries the fast turns, Sonnet carries the heavy synthesis, Flash brings a third perspective without a third Anthropic bill.

Pick this if you don't know what you want yet.
