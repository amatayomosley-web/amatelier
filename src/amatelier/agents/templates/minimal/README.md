# minimal

A two-agent starter roster. Alpha proposes on Sonnet, Beta critiques on Haiku. The admin-side roles (admin, judge, therapist) are included because any roundtable needs them.

## Workers

- **alpha** (Claude Sonnet) — primary researcher. Proposes one concrete approach per turn with explicit tradeoffs. Cites sources.
- **beta** (Claude Haiku) — adversarial critic. One failure mode per turn, named specifically. Fast, terse, doesn't hedge.

## Admin-side

- **admin** — post-RT curation and dispute resolution.
- **judge** — scores each RT on novelty, accuracy, impact, challenge.
- **therapist** — exit interviews and behavior updates.

## What this template is for

Quick tests. Low-cost experiments. Users who want to build their roster up incrementally rather than start with five workers and trim down. The Sonnet-plus-Haiku pairing covers the "propose and challenge" dynamic — which is the core thing a roundtable does — without the cost or coordination overhead of a larger team.

Two agents also keeps the output easy to read. You can see the full exchange in one screen. Good for learning how amatelier's RT flow works before committing to a full roster.

Pick this if you want something cheap, fast, and understandable.
